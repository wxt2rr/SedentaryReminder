## ✨ 核心功能

* **🕰️ 灵活的触发机制**
    * **固定间隔模式**：简单直接，例如“每隔 45 分钟提醒一次”。
    * **高级 Cron 模式**：支持标准的 5 位 Cron 表达式，实现复杂的提醒计划（如“仅在工作日的整点提醒”）。
* **📅 智能作息管理**
    * 设置你的**工作时段**（如 09:00-18:00），非工作时间自动静默。
    * 设置**午休屏蔽区**（如 12:00-14:00），午休期间不打扰。
* **🔔 多维度强力提醒**
    * **标准通知**：系统原生的通知横幅。
    * **动态状态栏**：菜单栏图标会在提醒时变成动画。
    * **屏幕中央弹窗** (可选)：一个醒目的动画图标在屏幕中央跳出。
    * **全屏强力遮罩** (可选)：强制覆盖全屏并模糊背景，点击“跳过”才能解除，专治各种不自觉。
* **💬 高度个性化**
    * 自定义通知的标题和内容文案。
    * 所有配置自动持久化保存。
* **🚀 极简体验**
    * 常驻菜单栏，不占用 Dock 栏空间。
    * 全新设计的现代化 UI，操作直观。
    * 原生 Swift 实现，体积小，资源占用几乎为零。

## 🛠️ 技术栈

* **语言**: Swift 5.x
* **框架**: SwiftUI, AppKit (用于窗口管理), Combine
* **平台**: macOS 13.0+ (Ventura 或更高版本)
* **架构**: MVVM (Model-View-ViewModel)
* **依赖**: 无任何第三方依赖，纯原生实现。

## 🚀 如何运行与安装

### 开发环境运行

1.  确保安装了最新版本的 **Xcode**。
2.  克隆或下载本项目代码。
3.  使用 Xcode 打开 `SedentaryReminder.xcodeproj` 文件。
4.  点击运行按钮 (Cmd + R)。

### 打包安装 (推荐)

1.  在 Xcode 中，选择顶部菜单栏的 `Product` -> `Archive`。
2.  在弹出的 Organizer 窗口中，点击右侧的 `Distribute App`。
3.  选择 `Custom` -> `Copy App`。
4.  将导出的 `.app` 文件拖入你的 `/Applications` (应用程序) 文件夹。
5.  **首次运行**：右键点击图标 -> 选择“打开” -> 在弹窗中点击“打开”以绕过安全检查。
6.  (可选) 在系统设置中将其添加为开机启动项。
